Cookbook Library

Cookbook is a tiered general-purpose library. It builds upon itself to create an extensive pool of functions to choose from. Cookbook contains functions ranging from highly generic (i.e, ([a] -> a -> [a] and [a] -> [a] -> [a] working on the same function) to very specific, like Quill2, the database language API with type safety.

Sound interesting? Here's some examples.

after "field:item" ':' => "item"

after "field:item" "d:" =>"item"`

splitOn "field,field,field" '.' => ["field","field","field"]

contains "field" 'e' => True

encompassing "field{field{field{x}}}" ('{','}') => "field{field{x}}"

import Cookbook.*.Meta => Import all modules in that branch

Get Cookbook

Cookbook can be installed easiest via the cabal command-line program. Hackage page

cabal install cookbook

It has a Debian package wrapper in the repository that will call the cabal program for you, for easy installation via gdebi or a similar program. If none of these options work, it can be installed manually by the clone URL.

Clone cookbook

git clone https://github.com/natepisarski/cookbook-hs
